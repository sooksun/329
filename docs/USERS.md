# บัญชีผู้ใช้ 329 MIS

## จำนวน

- **10 บัญชี** = `admin` + `director` + **8 คนฝ่ายงาน** (หัวหน้า + เจ้าหน้าที่ × 4 กลุ่มงานหลัก)
- รหัสผ่านเริ่มต้นฝ่ายงาน: **`Pass329!`**
- ผู้ใช้แก้ **ชื่อเต็ม / ชื่อผู้ใช้ / รหัสผ่าน** ได้ที่เมนู **โปรไฟล์ของฉัน** (`/profile`)
- Admin เพิ่มผู้ใช้ใหม่ได้ที่ **จัดการผู้ใช้** (`/admin/users`)

## บัญชีระบบ

| ชื่อผู้ใช้ | รหัสผ่าน | บทบาท |
|-----------|----------|--------|
| admin | admin123 | Super Admin |
| director | password123 | Project Director + Secretary |

## บัญชีฝ่ายงาน (4 กลุ่มงานหลัก)

| กลุ่มงาน (รวมจากเดิม) | หัวหน้า (username) | เจ้าหน้าที่ (username) |
|-----|-------------------|------------------------|
| อำนวยการและการเงิน <br/>(ภาพรวม + คณะกรรมการ + งบประมาณ) | gov_lead | gov_staff |
| กีฬา สถานที่ และความปลอดภัย <br/>(กีฬา + สถานที่ + ความปลอดภัย/แพทย์) | ops_lead | ops_staff |
| พิธีการ สวัสดิการ และประชาสัมพันธ์ <br/>(พิธีการ + อาหาร + PR + ทะเบียน) | host_lead | host_staff |
| เอกสารและติดตามงาน <br/>(เอกสาร/MIS + Timeline/ความเสี่ยง + ปฏิบัติการวันงาน + สรุปผล) | mis_lead | mis_staff |

> หมายเหตุ: กลุ่ม **อำนวยการและการเงิน** ได้สิทธิ์ `Finance Officer` (ดูแลงบประมาณ) เพิ่มให้อัตโนมัติ

รายชื่อเต็มอยู่ใน `prisma/user-seeds.ts`

## งานย่อย (Subtask)

- แต่ละงานย่อยมี **ผู้รับผิดชอบ** (`owner_id`) — คนเดียวรับผิดชอบได้หลายงานย่อยพร้อมกัน
- หัวหน้าฝ่าย / Admin กำหนดผู้รับผิดชอบได้จากหน้ารายละเอียดภารกิจ → แก้ไขงานย่อย
